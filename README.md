# RUN

## run on docker

```
cd ./myproject/docker/docker-compose.yml
docker-compose build 
docker-compose up -d

```
now porject is up on localhost:8081

## run on server

```
cd myproject
pip install -r requirements.txt
python3 runserver ./manage.py 

```

now porject is up on localhost:8000
